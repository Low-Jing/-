# 升级版前端说明

这是按“功能更丰富、界面更像产品、适合毕设答辩展示”的思路重做的 uni-app Vue2 前端起步包。

## 新增和强化点
- 从 4 个页面升级为 5 个页面：首页、计划、记录、发现、我的
- 首页加入仪表盘、快捷入口、身体概览、训练内容精选
- 计划页加入推荐算法阶段说明，便于论文与答辩解释
- 发现页加入课程专题、挑战和知识模块，用于拉高产品完整度
- 我的页面加入成长概览与更完整的打卡区域

## 使用方式
1. 在 HBuilderX 新建一个 uni-app Vue2 项目
2. 把本压缩包中的文件覆盖到项目根目录
3. 保持 Django 后端运行
4. 运行到浏览器调试

## 注意
- 如果你的后端还没有 /plans/exercises/ 和 /plans/foods/ 接口，计划页会自动使用本地演示数据，不会白屏
- 如果你把我后面给你的后端补丁也一起应用，前端会更完整
